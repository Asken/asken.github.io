Class ComponentController

Component controller

This class is a singleton and should not be instantiated directly. Each component is automatically added to the controller when it is created and removed when the component is destroyed.

Use the methods for context menu and modal components to manage the current context menu and modal.

Example

async showWindow(): Promise<BaseComponent> {
   if (!this.el) {
     await this.render()
     Dom.getBody().append(this.el!)
   }
   Dom.els(this.el!).css({
     display: 'block'
   })
   controller.setCurrentModal(this)
   this.modalSubscription = pubsub.subscribe(
     'modal-closed',
     (component) => {
       if (component === this) {
         this.hide()
       }
     },
     this
   )
   return this
 }
